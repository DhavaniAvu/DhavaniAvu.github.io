Student-Athlete NIL Data Analysis

Project Title: Data-Driven Sponsorship Matching for Student-Athletes at SponsorPro

Objective

Developed an analytical pipeline to process student-athlete NIL (Name, Image, and Likeness) data, ensuring structured athlete profiles and optimized sponsorship matching using data-driven insights.

My Role

As a Data Analyst, I automated data extraction, preprocessing, and analytics to enhance sponsor-athlete matching. Implemented Power BI dashboards to provide actionable insights and support data-driven decision-making.

Project Workflow and Contributions

  • Data Extraction: Automated web scraping and entity extraction using Python (BeautifulSoup, pandas, requests), reducing manual effort by 50%.
  • Data Cleaning & Structuring: Processed and normalized athlete records using SQL, used pandas and numpy for cleaning data, resolved inconsistencies.
  • Predictive Analytics & Segmentation: Implemented clustering models (K-Means) and statistical techniques (T-tests, ANOVA) to group athlets for better sponsorship potential.
  • Visualization & Reporting: Developed Power BI dashboards to track sponsorship trends, athlete engagement, and sponsorship success metrics, improving targeting efficiency by 35%.
  • Data Integration & Automation: Integrated Google Sheets API for real-time data synchronization, improving data retrieval speed and accuracy.

Tools and Technologies Used

  • Programming Languages: Python
  • Data Processing Libraries: pandas, NumPy, BeautifulSoup, requests
  • Database & Query Optimization: SQL (PostgreSQL)
  • Visualization & Reporting: Power BI, Google Sheets API
  • Machine Learning & Statistics: K-Means Clustering, T-tests, ANOVA

Challenges Encountered

  • Data Aggregation: Merging multi-source data into structured formats while maintaining consistency across athlete profiles.
  • Scalability: Ensuring efficient data retrieval and visualization for athlete datasets.

Outcomes and Impact

The project streamlined sponsor-athlete matching, reducing manual profiling efforts by 50%, improving sponsorship targeting by 35%, and ensuring high data accuracy for decision-making.
