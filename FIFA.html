FIFA Dataset Analysis

Project Title: Comprehensive Analysis of FIFA 19 Player Data

Project Overview

This project involves a detailed analysis of the FIFA 19 dataset, encompassing player attributes, club performance, and game statistics. The primary aim is to derive insights into player performance metrics, transfer values, and potential factors influencing player ratings. The analysis integrates advanced data manipulation techniques, statistical testing, and machine learning models to explore key patterns and predictions.

Research Questions

  • What are the main attributes that influence player ratings in FIFA 19?
  • How can we predict a player's transfer value based on their attributes?
  • Are there significant differences in player attributes across different positions or leagues?
  • What factors are most predictive of a player's overall rating?

Tools and Technologies

  • Programming Languages: Python, R
  • Data Manipulation: pandas, numpy
  • Data Visualization: matplotlib, seaborn, Tableau
  • Machine Learning: sklearn, xgboost
  • Statistical Analysis: scipy, statsmodels
  • Dimensionality Reduction: PCA

Methodology

The project employs a variety of techniques to analyze the dataset:

  • Data Cleaning: Addressed missing values, outliers, and inconsistent data entries using pandas and custom preprocessing scripts.
  • Exploratory Data Analysis (EDA): Utilized seaborn and matplotlib for visual exploration of data distributions, correlations, and patterns.
  • Feature Engineering: Created new features based on player attributes and performance metrics to enhance model accuracy.
  • Principal Component Analysis (PCA): Applied PCA to reduce dimensionality and identify the principal components influencing player ratings.
  • Regression Models: Implemented linear and polynomial regression models to predict player transfer values and ratings.
  • Machine Learning Models: Used decision trees and XGBoost for classification tasks and feature importance evaluation.

Results and Insights

The analysis provided the following key insights:

  • Key Influencers: Identified significant attributes affecting player ratings, such as player position, skill attributes, and age.
  • Predictive Modeling: Developed models with high predictive accuracy for player transfer values and ratings, demonstrating the impact of various attributes.
  • Position and League Analysis: Revealed notable differences in attributes across positions and leagues, influencing overall performance metrics.
  • Principal Components: PCA highlighted critical features that contribute most to player ratings, streamlining future analyses.
