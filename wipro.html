License Management and Security Log Analysis

Project Title: License Management and Security Log Analysis for Multi-Project Compliance and Efficiency at Wipro

Objective

The project aimed to create a centralized system for managing software licenses across Wipro's multiple projects. This system was designed to reduce costs, improve compliance with software vendor agreements, and streamline license allocation based on demand.

My Role

My role involved extracting license data from various sources and developing ETL (Extract, Transform, Load) processes to prepare the data for analysis and visualization. I handled data retrieval, transformation, and loading into a centralized repository for comprehensive analysis.

Project Workflow and Contributions

  • Data Extraction: Retrieved data from license servers, project management systems, and application logs. Used Pythonâ€™s re library for parsing logs and requests for API data extraction.
  • Data Transformation: Applied data cleaning and normalization with pandas, standardized formats, and created a common schema across all sources.
  • Loading Process: Loaded transformed data into a SQL database for analysis, implementing incremental loading to maintain data freshness without full re-imports.

Tools and Technologies Used

  • Programming Languages: Python, SQL
  • Data Processing Libraries: pandas, re, requests
  • Visualization: Tableau, matplotlib, seaborn
  • ETL Automation: Python scripts for scheduled tasks

Challenges Encountered

  • Data Complexity: Ensuring consistency across varied formats (unstructured logs, structured databases) required extensive cleaning and normalization.
  • Handling Large Volumes: Optimized log parsing and batched processing to manage high volumes of log data.

Outcomes and Impact

The project led to significant cost savings through license reallocation and ensured compliance with software agreements, reducing the risk of penalties. Our ETL pipeline provided timely, accurate insights for decision-making by Wipro's project managers and leadership.
