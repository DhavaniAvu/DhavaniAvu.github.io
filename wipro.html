License Management and Security Log Analysis

Project Title: License Compliance & Security Log Analysis for Wipro Internal Projects

Objective

Developed a centralized system for managing software licenses across Wipro projects using Python, reducing manual tracking efforts by 60% and optimizing cost reduction & compliance.

My Role

As a Data Analyst, I implemented ETL pipelines to extract, preprocess, and analyze software license data, ensuring compliance and optimizing software tracking through visualization dashboards.

Project Workflow and Contributions

  • Data Extraction: Retrieved and processed software license logs using Python (pandas, re), automating the extraction process for multiple internal projects.
  • Data Preprocessing & Cleaning: Processed and standardized 1M+ records using pandas & openpyxl, ensuring 99.9% data accuracy for compliance audits.
  • Validation & Automation: Integrated validation checks using Pytest for automated ETL functions
  • Visualization & Reporting: Designed Power BI dashboards to track license usage trends, compliance violations, and renewal alerts, improving tracking efficiency by 30%.
  • Compliance Monitoring: Collaborated with compliance and security teams to monitor unauthorized software usage, leading to a improvement in compliance enforcement.

Tools and Technologies Used

  • Programming Languages: Python
  • Data Processing Libraries: pandas, re, openpyxl
  • Visualization & Reporting: Power BI, matplotlib, seaborn
  • ETL Automation & Validation: Python scripts for log parsing, compliance validation, and report generation, integrated with Pytest

Challenges Encountered

  • Data Complexity: Standardizing varied log formats across multiple projects required extensive cleaning.
  • Real-time Compliance Tracking: Developed efficient ETL scripts to minimize processing time and ensure timely updates.

Outcomes and Impact

The project improved license tracking efficiency by 30%, reduced compliance violations by 40%, and optimized software cost allocation. The ETL automation reduced manual auditing time by 60%, allowing leadership to make informed decisions faster.
