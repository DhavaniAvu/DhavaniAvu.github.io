Student Performance Dashboard

Project Title: Interactive Student Performance Dashboard for Academic Insights

Objective

Developed an integrated student analytics system, enabling real-time visualization of student performance trends, department rankings, and failure analysis using Tableau, Google Sheets, and Firebase.

My Role

As a Business Intelligence Intern, I designed and deployed multiple dynamic Tableau dashboards, integrated them with MIT App Inventor, and optimized data retrieval through Google Sheets API and Firebase.

Project Workflow and Contributions

  • Dashboard Development: Designed and developed 5+ dynamic Tableau dashboards visualizing student performance trends, department rankings, and failure analysis.
  • Embedded Tableau in MIT App Inventor: Published dashboards on Tableau Server and dynamically integrated them into MIT App Inventor using WebView with student ID-based filtering.
  • Real-Time Data Integration: Implemented Google Sheets (Live Connection) with Tableau Server to enable real-time data updates without scheduled extracts.
  • Performance Optimization: Improved dashboard efficiency by pre-filtering data and using aggregated queries, reducing load times by 40%.
  • Role-Based Access Control: Configured Firebase Authentication to ensure faculty access was restricted to authorized dashboards based on department and role.

Tools and Technologies Used

  • Business Intelligence & Visualization: Tableau (Server & Public), WebView (MIT App Inventor)
  • Data Integration & APIs: Google Sheets API, Firebase Realtime Database
  • Data Processing: Google Sheets (Live Connection), Firebase Authentication
  • Optimization & Security: Role-Based Access Control, pre-filtering, and data aggregation techniques

Challenges Encountered

  • Data Synchronization: Ensuring real-time data updates while integrating Google Sheets API with Tableau Server.
  • Access Control: Implementing Firebase-based authentication to dynamically restrict access based on faculty roles.

Outcomes and Impact

The system streamlined academic performance tracking, reducing dashboard load time by 40% and improving data accuracy. The integration enabled faculty to filter and access student records dynamically, ensuring secure and efficient decision-making.
